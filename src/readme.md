# Special framework by php

I made framework by php
This is MVC model.
If you want to add database, you can use migration by phinx.
And this framework is used by Illuminate/database.
You can use Eloquent like Laravel.

## Getting Started

composer install

### Prerequisites

mysql 5.7
php 7.1
